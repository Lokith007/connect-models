# Twitter Stats Viewer

A simple full-stack application to fetch and display Twitter/X stats using a RapidAPI endpoint.

## Setup

1.  **Navigate to the server directory:**
    ```bash
    cd connect-models/twitter-stats/server
    ```

2.  **Dependencies are installed!**
    - I have already run `npm install` for you.

3.  **API Key Configured!**
    - I have updated `.env` with your API key.

4.  **Run the Server:**
    ```bash
    node index.js
    ```
    Server runs on `http://localhost:5000`.

5.  **Run the Client:**
    - Open `client/index.html` in your browser.
    - Or use a live server extension if preferred.

## Usage

- Paste a Tweet URL (e.g., `https://x.com/user/status/123...`) to get tweet stats.
- Paste a Profile URL (e.g., `https://x.com/user`) to get profile stats.
- Click "Get Stats".
